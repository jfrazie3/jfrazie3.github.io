Styles

To style an element, use the style attribute.

    Some things you can do with the style attribute include:

  • Background color- this is set in the start tag of the body element and defines the color of the page background, using (body style="background-color:color;") *note the lightblue background set in the body element
  • Text color- defines the color of text using (tag style="color:red;")
  • Font family- defines the font of an element using the (tag style="font-family:name;")
  • Font size- defines the size of font using a percent. (tag style="font-size:#%;")

  • Text align- defines where the content of an element will appear in the horizontal plane. (tag style="text-align:position;")

Formatting

To format content, use special elements to define how the content is displayed. To do this simply place the start and end tags of the special elements around the content you want to format.

    These special elements include:
  • Bold text using the (b) tags. This text is bold but has no extra importance.
  • Strong text using the (strong) tags. This text is bolded and has importance.
  • Italic text using the (i) tags. This text is italic with no extra importance.
  • Emphasized text using the (em) tags. This text is italics with extra importance.
  • Marked text using the (mark) tags. This text is highlighted
  • Small text using the (small) tags. This text appears small.
  • Deleted Deleted text using the (del) tags. This text has a line through it.
  • Inserted text using the (ins) tags. This text has a line below it.
  • Subscript text using the (sub) tags
  • Superscript text using the (sup) tags
