Styles

To style an element, use the style attribute.

    Some things you can do with the style attribute include:
  • Background color- this is set in the start tag of the body element and defines the color of the page background, using (body style="background-color:color;") *note the lightblue background set in the body element
  • Text color- defines the color of text using (tag style="color:red;")
  • Font family- defines the font of an element using the (tag style="font-family:name;")
  • Font size- defines the size of font using a percent. (tag style="font-size:#%;")
  • Text align- defines where the content of an element will appear in the horizontal plane. (tag style="text-align:position;")
