One is not sufficient without the other

The body contains all of the visible page content. This section is enclosed in a paragraph tag. *p*

Most tags come in pairs- The first tag is a start tag and the second tag is an end tag. *p*

A web browser reads the HTML documents and displays them according to the tags used.

When using a text editor, files can be saved as a .htm and the encoding should be set to UTF-8. These files can then be viewed in your browser by opening the file with a browser.

Tags

Tags are used to tell the browser how to display an item, such as text, images or links

Headings like this define large/bold text. They have the (h#) tag, and the number can range from 1 to 6 to change the size of the text.

Paragraphs are "normal text" and look like this. They use the (p) tag.

Links created using the (a href="https://www.website.com") tag.

Images like the one above can be inserted using the (img src="file.filetype" alt="Description of picture" width="width in pixels" height="height in pixels") tag. These extra parameters are known as attributes.

Lists can be created in HTML using either the (ul) unordered list tag or the (ol) ordered list tag. To create an item in a list you must use the (li) tag inside of an unordered or ordered list. An example of each is shown below.

Elements

An element consists of the start tag, content and end tag

Many elements are "nested" meaning they are contained inside other elements. Some exmples are the html element that contains the body or the paragraph elements which are the content of the body. Remember to include the end tag or your element may not function correctly.

An empty element, such as the line break (br), contain no content.

Attributes

Attributes provide additional information to the browser about the HTML elements.

  • All HTML elemnts can have attributes
  • Attributes provide additional information about an element
  • Attributes are specified in the start tag
  • Attributes are accessed using name-value pairs in the form (name="value")

Some useful attributes include

  • href Attribute- used to define links in the (a) tag. (a href="httpl://www.website.com") Some text (/a)
  • src Attribute- used to define the image source in the (img) tag. (img src="pic.jpg")
  • Width and Height Attributes- used to set the size of images in the (img) tag. (img src... width="# of pixels" height="# of pixels")
  • alt Attribute- defines text to be used as an alternative to the picture in the (img) tag. (img src... alt="Description of picture")
  • style Attribute- defines the styling of an element. (p style="color:red")
  • lang Attribute- declares a language in the (html) tag. (html lang="en-US")
  • title Attribute- displays a value as a tooltip (p title="test")
